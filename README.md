# fish-dl

[![Build Status][travis-badge]][travis-link]
[![Slack Room][slack-badge]][slack-link]

Wrapper around youtube-dl to easily download media when using the [fish shell].

## Install

With [fisherman]

```
fisher dideler/fish-dl
```

## Usage

```fish
fish-dl
```
